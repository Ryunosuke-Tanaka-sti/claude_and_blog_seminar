## 💡 使い分けガイド

### Mermaidを選ぶべきケース
- ✅ フローチャート、シーケンス図
- ✅ WordPressに直接埋め込みたい
- ✅ シンプルな図で十分
- ✅ テキストで管理したい

### SVGを選ぶべきケース
- ✅ 複雑なアイコン、ロゴ
- ✅ 拡大縮小が必要
- ✅ ベクター形式が必須
- ✅ Figmaで編集したい

### HTMLを選ぶべきケース
- ✅ 複雑なレイアウト
- ✅ Tailwind CSSが好き
- ✅ レスポンシブ対応
- ✅ 編集・修正が多い
